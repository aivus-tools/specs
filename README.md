## 🎯 Краткое описание проекта

**AIVUS** - платформа для создания и сравнения коммерческих предложений (estimation) между вендорами и клиентами.

### Основные роли:
- **VENDOR** - создает проекты, брифы, оферы (сметы)
- **CLIENT** - создает брифы, приглашает вендоров, сравнивает оферы

### Ключевые сущности:
- **Brief** - техническое задание (ТЗ) проекта
- **Project** - работа вендора над брифом
- **Offer** - коммерческое предложение (смета) к проекту

## 🛠️ Технологический стек

### Frontend
- **Next.js 14** (App Router)
- **TypeScript**
- **Redux Toolkit** + RTK Query
- **NextAuth.js** (сессии, JWT)

### Backend
- **Django 5.1** + Django REST Framework
- **PostgreSQL**
- **Docker** + Docker Compose

### Инфраструктура
- **HMAC Authentication** (между frontend и backend)
- **JWT Sessions** (NextAuth)
- **S3-compatible storage** (для файлов)
